# Análise de Complexidade dos Algoritmos

## BubbleSort
- Complexidade temporal:
    - Melhor caso: O(n) - quando já está ordenado (Percorre apenas 1 vez)
    - Caso médio: O(n²)
    - Pior caso: O(n²) - quando está ordenado inversamente
- Complexidade espacial: O(1) - utiliza espaço constante

## InsertionSort
- Complexidade temporal:
    - Melhor caso: O(n)
    - Caso médio: O(n²)
    - Pior caso: O(n²)
- Complexidade espacial: O(1)

## QuickSort
- Complexidade temporal:
    - Melhor caso: O(n log n)
    - Caso médio: O(n log n)
    - Pior caso: O(n²)
- Complexidade espacial: O(log n) - devido à recursão
